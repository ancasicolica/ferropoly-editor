    Verwendung privater Daten

    Für das Spiel werden die einen oder anderen privaten Daten (wie Namen, E-Mailadresse, Telefonnummer, ... ) verlangt. Diese Daten werden ausschliesslich für das Spiel verwendet und werden nicht an Dritte weitergegeben. Bei wichtigen Infos können die Email-Adressen von uns zur Kontaktaufnahme mit den Spielorganisatoren verwendet werden.

    Komerzielle Spiele

    Diese Software darf nur für nicht-komerzielle Spiele verwendet werden. D.h. der Organisator bezieht keinen Lohn oder Entschädigungen, welche über seine Auslagen hinausgehen. Für die Austragung komerzieller Spiele ist eine Vereinbarung mit dem Webseitenbetreiber zwingend. Fehlbare Organisatoren werden aus der Webseite ausgeschlossen.
